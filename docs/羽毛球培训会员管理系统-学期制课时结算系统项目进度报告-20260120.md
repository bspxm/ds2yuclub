## 一、项目概述

### 原始需求
将现有羽毛球培训系统的学员管理、课程管理等功能，整合升级为**完整的学期制课时结算系统**，核心需求包括：

- **多学期管理**：支持常规学期、暑假、寒假三种学期类型
- **课时包购买**：学员按学期购买固定次数的课程（1.5小时/次）
- **班级模式**：分为"固定天训练"和"家长自选"两种班级类型
- **考勤与补课**：教练登记考勤，请假不扣课时，自动顺延到下周同一时间补课
- **课时结转与信用系统**：学期结束剩余课时可结转，允许学员欠一定课时，建立课时信用系统
- **专注课程结算**：系统不涉及财务资金流，专注于课程次数的管理和结算

## 二、技术架构

### 后端技术栈 (FastAPI + SQLAlchemy)
- **框架**：FastAPI 0.115.2 + Uvicorn
- **ORM**：SQLAlchemy 2.0.45 + asyncmy (MySQL异步驱动)
- **数据验证**：Pydantic 2.0+
- **架构模式**：FastApiAdmin插件化架构（`module_badminton`独立模块）
- **分层架构**：Model → Schema → CRUD → Service → Controller

### 前端技术栈 (Vue3 + TypeScript)
- **框架**：Vue 3.5.17 + TypeScript 5.8.3
- **UI组件**：Element Plus 2.10.4
- **状态管理**：Pinia 3.0.3
- **HTTP客户端**：Axios 1.10.0
- **构建工具**：Vite 6.3.5

### 3. 缺失page_crud方法 ✅ 已修复
