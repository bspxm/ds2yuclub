# 羽毛球主题登录页重构设计

## 概述

将 `frontend/src/views/module_system/auth/index.vue` 登录页面从企业级管理系统风格重构为羽毛球比赛竞技运动风格。桌面端与移动端自适应适配。

## 设计风格

- **风格**：竞技运动风（Vibrant & Block-based）
- **视觉语言**：球场深绿 + 白色羽球 + 金色点缀
- **意象元素**：羽毛球剪影、抽象球场线条、羽球飞行轨迹弧线

## 色彩体系

| 用途 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 主色 Primary | `#1B5E20` | `#4CAF50` |
| 辅色 Secondary | `#388E3C` | `#66BB6A` |
| 强调色 Accent | `#F5A623` | `#FFD54F` |
| 页面背景 Background | `#F0F7F0` | `#0A1F0A` |
| 卡片背景 Surface | `rgba(255,255,255,0.92)` | `rgba(15,30,15,0.9)` |
| 正文文字 Text | `#1B5E20` / `#333333` | `#E8F5E9` / `#A5D6A7` |

## 装饰元素

1. **羽球 SVG 装饰** — 大尺寸白色半透明羽球剪影，浮动动画 (translateY 6px, 3s)
2. **抽象球场线条** — 中线/发球线，半透明，脉冲呼吸光效 (opacity, 4s)
3. **飞行轨迹弧线** — 从羽球延伸至表单区域的弧线，opacity 0.08
4. **品牌标语** — "挥拍之间，掌控全场"，Barlow Condensed 字体，金色

## 动效

| 元素 | 动效 | 时长 |
|------|------|------|
| 羽球装饰 | 浮动 (translateY) | 3s ease-in-out |
| 球场线条 | 呼吸光效 (opacity) | 4s ease-in-out |
| 登录卡片 | 入场滑入 + 缩放 | 0.7s ease |
| 输入框聚焦 | 绿色光晕边框 | 0.25s ease |
| 表单切换 | fade-slide (保持现有) | 0.35s |

## 字体

- **标题/标语**：Barlow Condensed (Google Fonts) — 竞技感、紧凑
- **正文/表单**：Barlow (Google Fonts) — 清晰易读
- Fallback：系统字体系列

## 组件结构

仅修改 `frontend/src/views/module_system/auth/index.vue` 一个文件。

- 桌面端左侧装饰区 + 右侧表单卡片
- 移动端顶部装饰区 + 底部表单卡片
- Login.vue / Register.vue / ResetPwd.vue 保持原样，作为动态组件渲染
- 工具栏（主题切换、语言切换）保持原样

## 移动端适配策略

- 使用 CSS `@media (max-width: 768px)` 切换布局
- 桌面端 grid 左右布局 → 移动端上下块状布局
- 移动端隐藏左侧装饰区，改为顶部深绿渐变区
- 输入框增大 touch target
- 不引入 Vant 组件，保持 Element Plus 但通过 CSS 适配移动端

## 实现边界

- 不改动 `Login.vue` / `Register.vue` / `ResetPwd.vue` 组件
- 不改动 `ThemeSwitch` / `LangSelect` / `CommonWrapper` 组件
- 不改动路由、store、权限逻辑
- 不改动 `router/index.ts` 或 `permission.store.ts`
- 保持所有现有功能（验证码、微信登录、记住我等）
- 保持所有 i18n 翻译
- 保持深浅色模式兼容
