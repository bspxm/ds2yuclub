# 时间段选择功能测试说明

## 功能概述

在新增购买记录（批量新增）时，根据班级类型自动处理时间段选择：

### 1. 固定班（FIXED）
- **行为**：自动全选所有可用时间段
- **用户操作**：无需手动选择，时间段复选框禁用
- **验证**：默认全选，不允许修改

### 2. 自选天（FLEXIBLE）
- **行为**：用户需要手动选择上课时间段
- **限制**：必须选择与班级总课时数相同的时间段数量
- **验证**：
  - 选择不足时显示警告："已选择 X 个时间段，还需选择 Y 个"
  - 选择超出时显示警告："已选择 X 个时间段，超出 Y 个"
  - 提交时验证是否选择了正确数量

### 2. 前端功能测试

#### 测试固定班
1. 登录系统
2. 进入"购买记录"页面
3. 点击"批量新增"按钮
4. 选择一个固定天类型的班级（class_type = "fixed"）
5. 观察时间段选择区域：
   - 自动加载该班级的所有排课记录
   - 所有时间段默认全选
   - 复选框禁用，不允许修改
   - 显示提示："已选择 X / Y 个时间段"

#### 测试自选天
1. 登录系统
2. 进入"购买记录"页面
3. 点击"批量新增"按钮
4. 选择一个自选天类型的班级（class_type = "flexible"）
5. 观察时间段选择区域：
   - 自动加载该班级的所有排课记录
   - 时间段默认不选中
   - 显示提示："需选择 N 次"（N = 总课时数）
6. 尝试选择时间段：
   - 选择少于N个：显示警告"还需选择 X 个"
   - 选择超过N个：显示警告"超出 X 个"
   - 选择等于N个：警告消失
7. 尝试提交：
   - 选择不足：提交失败，提示错误
   - 选择正确：提交成功

```sql
-- 为固定班创建排课（12次课）
INSERT INTO badminton_class_schedule (class_id, schedule_date, day_of_week, start_time, end_time, duration_minutes, coach_id, location, schedule_status)
SELECT 
    (SELECT id FROM badminton_class WHERE name = '测试固定班'),

    1, -- 周一
    '18:00:00',
    '19:30:00',
    90,
    1,
    '场地1',
    'scheduled'

    SELECT 0 as n UNION SELECT 1 UNION SELECT 2 UNION SELECT 3 UNION 
    SELECT 4 UNION SELECT 5 UNION SELECT 6 UNION SELECT 7 UNION 
    SELECT 8 UNION SELECT 9 UNION SELECT 10 UNION SELECT 11
) as numbers;

-- 为自选班创建多个可选时间段
INSERT INTO badminton_class_schedule (class_id, schedule_date, day_of_week, start_time, end_time, duration_minutes, coach_id, location, schedule_status)
SELECT 
    (SELECT id FROM badminton_class WHERE name = '测试自选班'),
    DATE_ADD('2026-01-27', INTERVAL n DAY), -- 连续多天
    (DAYOFWEEK(DATE_ADD('2026-01-27', INTERVAL n DAY)) - 1) % 7, -- 星期几

        WHEN n % 2 = 0 THEN '18:00:00'
        ELSE '19:00:00'

        ELSE '20:30:00'
    END,

    1,
    '场地2',

FROM (
    SELECT 0 as n UNION SELECT 1 UNION SELECT 2 UNION SELECT 3 UNION 
    SELECT 4 UNION SELECT 5 UNION SELECT 6 UNION SELECT 7 UNION 
    SELECT 8 UNION SELECT 9 UNION SELECT 10 UNION SELECT 11 UNION 
    SELECT 12 UNION SELECT 13 UNION SELECT 14

```

## 注意事项

1. **班级类型区分**：
   - 固定班（fixed）：每周固定时间上课，必须全选
   - 自选天（flexible）：用户自由选择上课时间，需满足课次要求

2. **验证逻辑**：
   - 固定班：自动全选，不允许修改
   - 自选天：必须选择与总课时数相等的时间段数量

3. **用户体验**：
   - 实时显示已选择数量
   - 实时显示选择不足或超出的警告
   - 提交前进行最终验证

4. **数据完整性**：
   - 确保排课记录存在且状态正确
   - 确保班级总课时数正确
