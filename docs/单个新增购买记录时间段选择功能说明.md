# 单个新增购买记录时间段选择功能说明

## 功能概述

为"新增购买记录"对话框添加了时间段选择功能，与"批量新增"功能保持一致。

## 实现内容

### 1. 前端修改

#### 文件：`frontend/src/api/module_badminton/purchase.ts`

- 在 `PurchaseForm` 接口中添加了 `selected_time_slots?: number[]` 字段

#### 文件：`frontend/src/views/module_badminton/purchase/index.vue`

##### 新增响应式变量
```typescript
// 单个新增表单相关
const availableTimeSlotsSingle = ref([]);
const classTypeInfoSingle = ref<any>(null);
const loadingTimeSlotsSingle = ref(false);
const timeSlotWarningSingle = ref("");
```

##### 新增辅助函数
1. `getUniqueDaysSingle()`: 获取所有不重复的星期
2. `getSlotsByDaySingle(day: string)`: 获取指定星期的时间段，按时间排序
3. `handleTimeSlotChangeSingle(selectedSlots: any[])`: 处理时间段选择变化
4. `handleClassChangeSingle(classId: number)`: 处理班级选择变化，加载可用时间段

##### 修改现有函数
1. `watch(() => formData.class_id, ...)`: 添加了对 `handleClassChangeSingle` 的调用
2. `resetForm()`: 添加了对单个新增表单相关变量的重置

##### 新增表单验证规则
```typescript
selected_time_slots: [
  {
    validator: (rule: any, value: number[], callback: any) => {
      if (!classTypeInfoSingle.value) {
        callback();
        return;
      }
      const requiredCount = classTypeInfoSingle.value.sessions_per_week || 0;
      if (classTypeInfoSingle.value.class_type === 'flexible') {
        if (!value || value.length === 0) {
          callback(new Error("请选择上课时间段"));
        } else if (value.length !== requiredCount) {
          callback(new Error(`请选择 ${requiredCount} 个上课时间段`));
        } else {
          callback();
        }
      } else {
        callback();
      }
    },
    trigger: "change"
  }
]
```

##### 新增UI模板
在"上课时间段"表单项中添加了时间段选择UI，包括：
- 未选择班级时的提示
- 加载中的提示
- 无可用时间段的提示
- 时间段选择复选框（按星期分组显示）
- 选择数量警告提示

### 2. 功能特性

#### 固定班（fixed）
- 自动全选所有可用时间段
- 禁用用户修改
- 不显示警告提示

#### 自选天（flexible）
- 用户需要手动选择时间段
- 根据每周课次（`sessions_per_week`）进行验证
- 实时显示选择状态（已选择X个，还需选择Y个）
- 表单验证确保选择数量正确

#### UI设计
- 采用深色主题配色（#1e1e1e, #252526, #3e3e42, #4c4d4f）
- 按星期分组显示时间段
- 每天的时间段按时间排序
- 支持滚动查看（最大高度300px）

### 3. 数据流程

1. 用户打开"新增购买记录"对话框
2. 选择班级后，自动调用 `handleClassChangeSingle`
3. 后端返回班级可用时间段和班级类型信息
4. 根据班级类型自动选择或清空时间段
5. 用户可以修改选择（如果是自选天）
6. 表单验证确保选择数量正确
7. 提交表单时，`selected_time_slots` 字段包含选择的时间段ID数组

### 4. 测试步骤

1. 打开浏览器访问 `http://localhost:5180/web#/Badminton/purchase`
2. 点击"新增购买记录"按钮
3. 选择一个固定班
   - 验证：所有时间段自动全选，且禁用修改
4. 选择一个自选天班级
   - 验证：时间段未选中，显示需要选择的数量
   - 选择不足数量时，显示警告
   - 选择超过数量时，显示警告
   - 选择正确数量时，警告消失
5. 提交表单验证
   - 未选择时间段时，应显示验证错误
   - 选择数量不正确时，应显示验证错误

### 5. 注意事项

- 该功能依赖于后端API `GET /badminton/classes/{class_id}/available-time-slots`
- 必须先选择班级才能显示时间段
- 重置表单时会清空所有时间段相关数据
- 编辑购买记录时，如果班级类型为固定，时间段会自动全选
